## Introduction to Python (1991)

- _Python is a high-level, general-purpose, and popular programming language._
- _It was created by Guido van Rossum and released in 1991._
- _Python is used for:_
  - _Web Development (Server-Side)_
  - _Software Development_
  - _Machine Learning Applications_
  - _Mathematics_
  - _System Scripting_
  - _Game Development_
  - _Artificial Intelligence (AI)_
  - _Big Data_
  - _Data Science_
  - _Internet of Things (IoT)_
  - _Deep Learning_

### Features of Python

- _Python is platform-independent and works on different platforms:_
  - _Windows_
  - _Linux_
  - _macOS_
  - _Raspberry Pi_
- _Python programs are generally shorter than programs written in many other languages._
- _Very simple syntax and easy to learn._
- _Python supports multiple programming paradigms:_
  - _Object-Oriented Programming (OOP) like Java_
  - _Procedural Programming like C_
  - _Functional Programming_
- _Python has automatic garbage collection._
- _Python is dynamically typed; you do not need to declare variable types explicitly._
- _The standard file extension for Python files is `.py`._

# Python Syntax

## Python Indentation

- _Indentation refers to the spaces at the beginning of a code line._
- _In many programming languages, indentation is used mainly for readability._
- _In Python, indentation is used to define a block of code._
- _Python raises an error if indentation is skipped where required._
- _The number of spaces is up to the programmer, but **4 spaces** is the most common convention._
- _The entire block of code should use the same number of spaces; otherwise, an error may occur._

## Python Variables

- _In Python, variables are created when you assign a value to them._
- _Python does not require a command for variable declaration._

### Example

```python
x = 143
```

## Comments in Python

- _Comments are used to explain code and make it more readable._
- _Python provides commenting capabilities for in-code documentation._
- _A comment starts with the `#` symbol, and Python ignores everything after it on that line._

### Example

```python
# This is a comment
```

### Types of Comments

#### 1. Single-Line Comment

- _Starts with the `#` symbol._
- _Anything after `#` is ignored by Python._

Example:

```python
# This is a single-line comment
```

#### 2. Multi-Line Comment

- _Python does not have a dedicated multi-line comment syntax._
- _Triple quotes (`'''` or `"""`) are commonly used as a workaround._

_Example:_

```python
'''
Python is beautiful at the beginning
Python remains beautiful at the end of the journey.
'''
```

_or_

```python
"""
Python is beautiful at the beginning
Python remains beautiful at the end of the journey.
"""
```
